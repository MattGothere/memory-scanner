# Memory Scanner

A personal project to understand how Cheat Engine works by building a simple memory scanner from scratch.

## Goals

- [ ] List running processes
- [ ] Open a target process
- [ ] Read process memory
- [ ] Enumerate memory regions
- [ ] Search for integer values
- [ ] Filter search results
- [ ] Build a simple command-line interface

## why

learn more about Windows internals, virtual memory, and security by implementing a simplified Cheat Engine.
